# 2048 Game — C++ Terminal Version

以 C++ 實作的終端機 2048 益智遊戲，採用物件導向設計，將遊戲邏輯與畫面渲染分離為獨立的類別。

## 功能特色

- 4×4 棋盤，終端機即時渲染
- 每次移動後隨機生成新數字（2 或 4）
- 自動判定勝利（達到 2048）與結束（無法移動）
- 支援鍵盤操作：`W A S D`
- 支援 Windows 終端機 Unicode 顯示

---

## 程式架構

本專案採用物件導向設計，將不同職責清楚分離：

| 類別 | 負責內容 |
|------|---------|
| `Game2048` | 核心遊戲邏輯 — 棋盤狀態、移動處理、合併規則、勝負判定 |
| `Display` | 畫面渲染 — 終端機顯示與遊戲主迴圈控制 |

```
main.cpp
  └── Display（負責畫面渲染）
        └── Game2048（負責遊戲狀態）
```

## 如何執行

### 方法一：線上試玩

點上方 **線上試玩 Demo** 連結，直接在瀏覽器玩，無需安裝。

### 方法二：下載執行檔

點上方 **下載 Windows 執行檔**，下載 `2048.exe` 後直接點兩下執行。

### 方法三：自行編譯

**環境需求**
- g++ 編譯器（C++11 以上）
- Windows（使用 MinGW）

**編譯與執行**

```bash
git clone https://github.com/youzhen0827/Game-2048.git
cd Game-2048
g++ main.cpp Game2048.cpp Diaplay.cpp -o 2048.exe
2048.exe
```

## 操作說明

| 按鍵 | 動作 |
|------|------|
| `W` | 向上移動 ↑ |
| `S` | 向下移動 ↓ |
| `A` | 向左移動 ← |
| `D` | 向右移動 → |
| `Esc` | 離開遊戲 |

- 相同數字的方塊碰在一起會**合併**
- 每次移動後會隨機出現一個新方塊
- 成功合出 **2048** 即獲勝
- 棋盤填滿且無法移動時遊戲結束
